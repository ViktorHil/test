   Программа на Microsoft Visual Basic (VB6), управляющая светодиодом на макетной плате AVR-USB-MEGA16, любезно предоставленная Степаненко Вячеславом (Казахстан, г. Талдыкорган). Работает точно так же, как и GUI-программа, написанная на Microsoft Visual Stidio C++ (.NET 2003), см. статью "Как использовать библиотеку libusb в Visual Studio (управление макеткой AVR-USB-MEGA16)" http://microsin.ru/content/view/757/44/.
   
[Ссылки]

1. Visual Basic 6.0 - How to access the libusb (http://libusb.6.n5.nabble.com/Visual-Basic-6-0-How-to-access-the-libusb-td10562.html).
2. LIBusb interface code (http://read.pudn.com/downloads105/doc/431308/libusb.bas__.htm)
3. http://www.uploadarchief.net/files/download/USB%20vb.zip - обертка для функций libusb, выполненная в виде libusbvb0.dll, с исходниками (GCC). Позволяет программе на Visual Basic VB6 получать доступ к функциям библиотеки libusb.
